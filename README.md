# MLSysBook Educational Resources

A simple, clean website for Machine Learning Systems educational resources.

## About

This repository contains a simple static website showcasing educational resources for machine learning systems, including:

- Open access textbook
- TinyTorch framework
- Hardware kits
- Instructor materials
- Community resources

## Deployment

This site is deployed on GitHub Pages and available at [mlsysbook.org](https://mlsysbook.org).

## Development

Simply edit `index.html` to update the website content. The site uses vanilla HTML/CSS with no build process required.
